### InstanceName Property (InstallSitePartSetup)

Instance Name of the Site Part

Syntax

```vbnet
'Declaration

<PropertyBitValueAttribute()>
Public Property InstanceName As String
```

```csharp
[PropertyBitValueAttribute()]
public string InstanceName {get; set;}
```

#### Remarks

If this is left blank, the Instance Name is set to the PArt Number and the Part Revision of the Site Part.
