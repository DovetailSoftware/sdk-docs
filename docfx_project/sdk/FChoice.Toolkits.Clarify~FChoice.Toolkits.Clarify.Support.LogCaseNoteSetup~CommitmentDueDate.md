### CommitmentDueDate Property (LogCaseNoteSetup)

DateTime when the commitment is due. If not set, the LogDate is used.

Syntax

```vbnet
'Declaration

<PropertyBitValueAttribute()>
Public Property CommitmentDueDate As Date
```

```csharp
[PropertyBitValueAttribute()]
public DateTime CommitmentDueDate {get; set;}
```

#### Remarks
