### PriceProgram Property (CreateContractSetup)

The title of the Price Program for this Contract. If not set, then "US Std Price List" will be used.

#### Syntax

```vbnet
'Declaration

<PropertyBitValueAttribute()>
Public Property PriceProgram As String
```
